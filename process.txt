1. make folder frontend and backend
2. create frontend using vite for react
3. in chat-app run [npm init -y] not in backend 
    3.1 -> create server.js in backend and change the name in package.json ["main": "server.js",]

4. then install 
    4.1 -> in chat-app run [npm install express dotenv cookie-parser bcryptjs mongoose socket.io jsonwebtoken ]

    why all the above mentioned dependencies
    1. express -> for backend
    2. dotenv -> to read environment variable 
    3. cookie-parser -> to pass the cookie
    4. bcryptjs -> able to hash password 
    5. mongoose -> database
    6. socket.io -> for real time communication
    7. jsonwebtoken -> to create token


5. install nodemon in chat-app and replace the script in package.json [    "server": "nodemon backend/server.js"
]
